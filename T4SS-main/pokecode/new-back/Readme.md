### back

### 环境
- nodejs14
- mysql
- vscode

### 项目目录架构
```shell
.
├── Readme.md           # 说明文件
├── app.js              # 启动app
├── configs             # 数据库配置
├── controls            # 业务sql
├── node_modules        # 依赖包
├── package-lock.json
├── package.json
├── routes              # 接口定义
├── sql                 # sql工具
├── uploads             # 文件上传目录
├── utils               # 工具类
└── ws                  # websocket
```  

### 安装运行
- 首次运行安装依赖包
```shell
npm install -g nodemon
# 安装依赖包
npm install 
```
   
- 运行应用
```shell
npm run start
```
